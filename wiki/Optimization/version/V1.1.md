# V1.1

## 账户与登录

- 个人账户支持注销账户，并在执行前提供二次确认。
- 登录页面补充“首次登录时自动注册”的说明，减少登录和注册入口的理解成本。
- 登录成功和退出登录时提供更醒目的反馈。
- 修正登录界面中的错别字。

## 聊天与会话

- 新增会话改为乐观更新，侧栏会立即出现新会话；创建失败时自动回滚。
- 删除对话改为乐观更新，失败时自动回滚。
- 新对话会根据首条消息自动命名。
- 生成回复期间限制其它对话发送消息，避免并发状态互相影响。
- 支持 `Ctrl+N` 快捷键新建对话。

## 设置与提示

- 未填写 API Key 时，主页会提醒补充配置；URL 未填写时使用默认官方地址。
- 左下角头像菜单新增更新日志入口，方便随时查看版本变化。
- 登录后会提示本次更新内容；同一浏览器每天最多自动弹出一次，也可以选择不再提示本次更新。
